a realtime search box, where users search articles, and then to have analytics that display what users were searching for

## Key Features <a name="key-features"></a>


- register and log in, so that the data is private to them.
- Add New Article
- Search for an Article
- Save Search query for the analysis

## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

<!--
Example command:

```sh
 gem install rails
```
 -->

### Setup

Clone this repository to your desired folder:


```
  git clone https://github.com/Bateyjosue/search-box.git
  
  cd search-box
  
```

### Install

Install this project with:

```
  bundle install
```

### Usage

To run the project, execute the following command:
1. Create Database
```
  rails db:create
```

2. run the migration
```
  rails db:migrate
```

3. Then run the develelopment server
```
  rails server
```


### Run tests

To run tests, run the following command:


```
  rails rspec spec
```

### Deployment

The app was deployed on Render.com
